gitとは

ファイルを編集した際に編集履歴が残ることで複数人でファイルの変更状況がわかりやすく管理できる仕組みです。
新しいバージョンのファイルも古いバージョンのファイルもGitが管理するので編集前の古いバージョンのファイルへ戻しやすくなっています。

HP更新の手順 　私たちはHPの更新を他の人に共有するためにGit hubを利用しています。
HPの更新をGit hubに反映させるためにこのような手順を踏みます。 Macはターミナル、WindowsはGit Bashを利用します。
  1. コマンドを入力する。変更内容を全て反映します。
  2. コマンドを入力する。今回の変更の履歴を残します。
    そのため日付や変更箇所の記述を残すと見やすいです。
  3. pushコマンドを入力する。Git hubに送信します。
注意 Git hubに動画をあげてはいけません。 リポジトリが壊れてしまいます。 YouTubeのURLを貼ることで動画をあげましょう。

用語集
  • リモートリポジトリ
    • 　ファイルを複数人で共有するためのリポジトリです。
  • ローカルリポジトリ
    • 　利用者一人一人のパソコンにあるリポジトリです。
  • Git hub
    • Gitの仕組みを利用して他の利用者とのやりとりをしやすくしているWebのサービスです。
